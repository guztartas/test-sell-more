# Test-sell-more

## Tecnologias Usadas

- React.js 19.x
- Next.js 15.x
- Tailwind CSS v4
- ESLint & Prettier
- Zod
- Playwright

## Instalação

```bash
git clone <https://github.com/guztartas/test-sell-more>
cd test-sell-more
npm install
```

## Rodar o Projeto

```bash
npm run dev
```

## Rodar Testes

```bash
npx playwright test
```
